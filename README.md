## Установка

### Запуск проекта локально (Node JS, http://localhost:3000):
```
npm run start
```
### Запуск проекта локально (parcel, http://localhost:1234):
```
npm run dev
```
### Сборка проекта (в директорий build):
```
npm run build
```
